# Checklist da entrega final / prova

Antes de entregar no Classroom, confira:

- [ ] Arquivo final salvo como `dashboard_vendas_prova.pbix`
- [ ] Página `PROVA - Visão Executiva` criada no Power BI
- [ ] Página `MER - Modelo de Dados` criada no Power BI ou MER apresentado como imagem separada
- [ ] Cards principais criados
- [ ] Gráfico de faturamento por mês criado
- [ ] Gráfico de Top 10 produtos criado
- [ ] Gráfico de faturamento por região criado
- [ ] Tabela detalhada criada
- [ ] Filtros interativos criados
- [ ] Filtros testados
- [ ] MER gerado e legível
- [ ] Arquivos finais enviados ao GitHub
- [ ] Board atualizado
- [ ] Vídeo gravado
- [ ] Link do vídeo conferido
- [ ] Texto da entrega preenchido no Classroom

## Entrega no Classroom

Obrigatório enviar:

- Link do vídeo
- Link do board
- Link do GitHub

Além disso, o vídeo deve apresentar:

- A funcionalidade final
- O board
- O GitHub
- O MER
